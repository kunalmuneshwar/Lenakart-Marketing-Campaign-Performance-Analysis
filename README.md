#  Marketing Campaign Performance Analysis

##  Project Overview

This project analyzes the performance of paid marketing campaigns using Microsoft Excel / Google Sheets. It covers the complete analytics workflow, including data cleaning, KPI calculation, pivot table analysis, dashboard creation, and business recommendations to evaluate campaign effectiveness.

---

##  Objectives

- Clean and prepare the raw marketing dataset.
- Calculate key marketing performance metrics.
- Analyze campaign performance across different dimensions.
- Build an interactive dashboard for business reporting.
- Generate actionable insights and recommendations.

---

##  Tools Used

- Microsoft Excel / Google Sheets
- Pivot Tables
- Charts & Dashboard
- Advanced Excel Formulas

---

##  Project Workflow

### 1. Data Cleaning
- Preserved the original raw dataset.
- Standardized date and data formats.
- Removed currency symbols where required.
- Standardized text values and naming conventions.
- Handled formula errors using IFERROR().
- Created calculated KPI columns.

### 2. KPI Calculations
The following marketing KPIs were calculated:

- Click Through Rate (CTR)
- Cost Per Click (CPC)
- Landing Page View Rate (LPV Rate)
- Add-to-Cart Rate
- Purchase Conversion Rate
- Cost Per Acquisition (CPA)
- Return on Ad Spend (ROAS)
- Average Order Value (AOV)

### 3. Pivot Table Analysis

Created analytical reports for:

- Platform Performance (Weekly)
- Campaign Performance (Weekly)
- Campaign Performance
- Creative Performance
- Product Category Performance

### 4. Dashboard

Designed an executive dashboard containing:

- Total Spend
- Total Revenue
- Total Purchases
- Average ROAS
- Average CPA
- Revenue by Platform
- Purchases by Platform
- Weekly Revenue Trend
- Campaign Performance
- Product Category Performance

---

##  Key Insights

- Google campaigns generated the highest revenue and strongest ROAS.
- Brand Search campaigns delivered the best commercial performance.
- Week 4 recorded the highest overall campaign performance.
- Eyeglasses and Contact Lenses were the top-performing product categories.
- Some awareness campaigns showed high spend but comparatively low returns.

---

##  Recommendations

- Increase investment in high-performing campaigns.
- Reduce spending on consistently low ROAS campaigns.
- Continue optimizing top-performing product categories.
- Perform A/B testing on underperforming creatives.
- Monitor campaign performance weekly for continuous optimization.
